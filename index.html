Welcome to Proxy UI

Proxy UI is a Roblox Luau UI Library designed for making polished, responsive interfaces. It supports scaling for mobile and PC, a built-in notification system, config saving, and custom theming.

Parameters

Name (string): Title of the window
Keybind (string): Key to toggle UI
Logo (string): Create your logo publish into roblox then paste your id after :// in rbxassetid
Scale (enum or UDim2): UI scale (mobile, window, or custom)
TextSize (number): Default text size

Notifications

Create notifications

local Notifier = ProxyUI.newNotify()

Notifier.new({
    Title = "Notification",
    Content = "Thank you for using this script!",
    Duration = 10,
    Icon = "rbxassetid://"
})

Parameters

Title (string): Title of the notification
Content (string): Message displayed
Duration (number): How long it stays on screen
Icon (string): Same Thing said in Windows Tab

Config Manager

Manage saving and loading of configs.

local ConfigManager = CompKiller:ConfigManager({
    Directory = "My-UI", 
    Config = "MyConfigs"
})

local ConfigUI = Window:DrawConfig({
    Name = "Config",
    Icon = "folder",
    Config = ConfigManager
})

ConfigUI:Init();

Toggles

Toggles allow users to enable/disable features.

local Toggle = Section:AddToggle({
    Name = "Godmode",
    Flag = "Toggle_Godmode",
    Default = false,
    Callback = function(state)
        print("Godmode:", state)
    end
})

Linking a Keybind

local Keybind = Toggle.Link:AddKeybind({
    Default = "E",
    Flag = "Godmode_Keybind",
    Callback = print
})

Sliders

Sliders let users pick a value between a range.

Section:AddSlider({
    Name = "WalkSpeed",
    Min = 0,
    Max = 100,
    Default = 16,
    Round = 0,
    Flag = "Slider_WalkSpeed",
    Callback = function(value)
        print("WalkSpeed set to:", value)
    end
})

Parameters

Name (string): Label for the slider
Min (number): Minimum value
Max (number): Maximum value
Default (number): Default value
Round (number): Decimal rounding (0 = integer)
Flag (string): Save ID for config
Callback (function): Function when value changes

Dropdowns

Dropdowns let users select one or multiple options.

Single Dropdown

Section:AddDropdown({
    Name = "Select Body Part",
    Default = "Head",
    Flag = "Single_Dropdown",
    Values = {"Head","Body","Arms","Legs"},
    Callback = print
})

Multi Dropdown

Section:AddDropdown({
    Name = "Select Multiple",
    Default = {"Head"},
    Multi = true,
    Flag = "Multi_Dropdown",
    Values = {"Head","Body","Arms","Legs"},
    Callback = print
})

Buttons

Buttons let users trigger actions.

Section:AddButton({
    Name = "Teleport",
    Callback = function()
        print("Teleporting...")
    end
})

Textboxes

Textboxes let users input text.

Section:AddTextBox({
    Name = "Custom Name",
    Placeholder = "Enter text...",
    Default = "Player1",
    Callback = print
})

Color Pickers

Let users choose a color.

Section:AddColorPicker({
    Name = "Highlight",
    Default = Color3.fromRGB(0, 255, 140),
    Flag = "Color_Highlight",
    Callback = print
})

Paragraphs

Paragraphs are blocks of text inside your UI.

Section:AddParagraph({
    Title = "Info",
    Content = "This is a description."
})

Themes

Change the look of your UI with themes.

ThemeTab:DrawSection({
    Name = "UI Themes"
}):AddDropdown({
    Name = "Select Theme",
    Default = "Default",
    Values = {
        "Default",
        "Dark Green",
        "Dark Blue",
        "Purple Rose",
        "Skeet"
    },
    Callback = function(v)
        ProxyUI:SetTheme(v)
    end
})

Watermark

The watermark appears at the corner of your UI.

local Watermark = Window:Watermark()

Watermark:AddText({
    Icon = "user",
    Text = "User123"
})

Watermark:AddText({
    Icon = "clock",
    Text = ProxyUI:GetDate()
})
